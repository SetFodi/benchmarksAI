# AndMarks | Frontier AI Benchmarks

AndMarks is a premium AI benchmark dashboard providing real-time and verified leaderboards for frontier AI models. Track the evolution of intelligence across multiple domains including general reasoning, coding, and UI generation.

![AndMarks Dashboard](public/globe.svg)

## Features

- **Live OpenRouter Integration**: Real-time context window data fetched directly from the OpenRouter API.
- **Verified Leaderboards**: Curated data from top-tier benchmarks like LMSYS Chatbot Arena, Design2Code, and SWE-bench Verified.
- **Premium UI/UX**: Compact, high-end dashboard design with glassmorphism, ambient lighting, and interactive data visualizations.
- **Dual View Modes**: Switch between detailed list views and visual intelligence distribution graphs.
- **Mobile Optimized**: Fully responsive design for tracking benchmarks on the go.

## Benchmarks Included

1.  **Chatbot Arena (Elo)**: The definitive crowdsourced leaderboard based on human preference.
2.  **LiveCodeBench**: Holistic competitive programming and coding evaluation.
3.  **GPQA (Science)**: Graduate-level reasoning across biology, physics, and chemistry.
4.  **Design2Code (UI)**: State-of-the-art evaluation for converting visual designs into frontend code.
5.  **SWE-bench Verified**: Real-world software engineering issue resolution performance.
6.  **Model Context**: Real-time context capabilities of the newest frontier releases.

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/andmarks.git
   cd andmarks
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

The easiest way to deploy AndMarks is via [Vercel](https://vercel.com):

1. Push your code to a GitHub repository.
2. Connect the repository to Vercel.
3. Vercel will automatically detect Next.js and deploy your dashboard.
